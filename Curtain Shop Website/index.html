MagazineleJeorjette sunt un partener competent pentru cei ce doresc sa-si decoreze caminul.

Perdelele si draperiile din colectiile noastre asortate si accesorizate modern dau ambientului originalitate, intimitate si oglindesc personalitatea fiecaruia.
